Spotify Clone

This project is a simple Spotify-inspired music player web application developed using HTML, CSS, and JavaScript.

Description
The Spotify Clone allows users to play audio files through a clean and user-friendly interface. It includes basic music player controls and demonstrates core front-end development concepts.

Features
- Play and pause audio
- Next and previous song controls
- Progress bar with current time and total duration
- Simple and responsive user interface

Technologies Used
- HTML
- CSS
- JavaScript

Purpose of the Project:
This project was created to practice front-end web development and JavaScript DOM manipulation concepts.
